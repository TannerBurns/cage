# Poker House Management Project

    Cage - Database management API for MAPP

# How To

    1. Get Golang dependencies found in requirements.txt

    2. Run setup.bat found inside of cloned directory (Windows), go build cage (Other)


# Current features
    
    Management:

        add players
        read players
        update players
        
        add membership
        read membership
        update membership

        add employee
        read employee
        updated employee
        
        add roles
        read roles
        update roles
        
        add permissions
        remove permissions
        
        add compensations
        
        log transactions
        log checkin/checkout
        
    API:
        
        Login
        Authentication (JWT Token)
        Privileges
        
        GET,POST,PUT methods
        Custom routes for MAPP
